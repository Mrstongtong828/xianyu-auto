# 飞书聊一聊（议价）功能使用指南

## 功能简介

飞书聊一聊功能允许你在飞书群聊中通过@机器人发送闲鱼商品链接，系统会自动使用指定的闲鱼账号向卖家发送议价消息。适用于批量询价、监控竞品价格等场景。

## 功能特点

- **便捷操作**：在飞书中@机器人即可发起议价
- **智能解析**：自动识别闲鱼商品链接
- **灵活话术**：支持自定义每条消息的议价内容
- **防封保护**：内置多重防封机制，模拟真人行为
- **频率控制**：支持配置发送频率限制

## 快速开始

### 第一步：创建飞书应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 登录后点击右上角"创建应用"
3. 选择"创建企业自建应用"
4. 填写应用信息：
   - **应用名称**：闲鱼议价助手（或其他你喜欢的名称）
   - **应用描述**：自动向闲鱼卖家发送议价消息
   - **应用图标**：上传一个图标（可选）
5. 点击"创建"

### 第二步：获取应用凭证

1. 进入应用详情页
2. 点击左侧菜单"凭证与基础信息"
3. 复制以下信息：
   - **App ID**（应用 ID）
   - **App Secret**（应用密钥）
4. 将这些信息填入本系统的"飞书聊一聊"配置页面

### 第三步：开启机器人功能

1. 在应用详情页，点击左侧"机器人"菜单
2. 打开"启用机器人"开关
3. 设置机器人名称和描述
4. 点击"保存"

### 第四步：配置事件订阅

1. 点击左侧"事件与回调" → "事件订阅"
2. **配置加密策略**（推荐）：
   - 打开"加密策略"开关
   - 系统会自动生成 Encrypt Key
   - 复制 Encrypt Key 填入本系统配置
3. **配置请求地址**：
   - 在"请求地址"栏填入：`http://你的服务器地址:端口/feishu/webhook`
   - 例如：`http://192.168.1.100:8090/feishu/webhook`
   - 如果是公网服务器，使用公网 IP 或域名
4. **订阅事件**：
   - 点击"添加事件"
   - 搜索并添加"接收消息"事件（im.message.receive_v1）
5. 点击"保存"

### 第五步：发布应用

1. 点击左侧"版本管理与发布"
2. 点击"创建版本"
3. 填写版本信息：
   - **版本号**：1.0.0
   - **更新说明**：初始版本
4. 点击"保存"
5. 点击"申请发布"
6. 等待管理员审核通过（如果是企业自建应用，通常自动通过）

### 第六步：配置本系统

1. 登录闲鱼管理系统
2. 进入"系统设置"页面
3. 找到"飞书聊一聊（议价）"配置卡片
4. 填写以下信息：
   - **App ID**：从飞书应用详情页复制
   - **App Secret**：从飞书应用详情页复制
   - **Encrypt Key**：从飞书事件订阅页面复制（如果启用了加密）
   - **议价账号**：选择用于发送议价消息的闲鱼账号
   - **默认议价话术**：设置默认的议价消息内容
5. 根据需要调整防封保护设置
6. 点击"保存配置"
7. 点击"测试连接"验证配置是否正确

### 第七步：添加机器人到群聊

1. 在飞书中打开需要使用议价功能的群聊
2. 点击群设置 → 群机器人 → 添加机器人
3. 搜索并选择你创建的应用机器人
4. 点击"添加"

## 使用方法

### 基本用法

在配置了机器人的群聊中发送消息：

```
@闲鱼议价助手 https://m.goofish.com/item?id=1234567890
```

系统会自动使用配置的闲鱼账号向该商品的卖家发送默认议价话术。

### 自定义议价内容

在链接后面添加你想要的议价内容：

```
@闲鱼议价助手 https://m.goofish.com/item?id=1234567890 老板，这个能便宜50吗？
```

系统会发送你指定的内容："老板，这个能便宜50吗？"

### 支持的链接格式

系统支持多种闲鱼链接格式：

- `https://m.goofish.com/item?id=1234567890`
- `https://www.goofish.com/item?id=1234567890`
- `https://m.tb.cn/h.xxxxxx`（淘宝短链接）
- 包含商品ID的任何文本

## 防封保护机制

为了保护你的闲鱼账号不被封禁，系统内置了多重防封机制：

### 1. 随机延迟

进入聊天框后，系统会随机等待 2-5 秒（可配置），模拟真人阅读商品信息的时间。

### 2. 模拟打字

系统会根据消息长度模拟打字时间：
- 短消息（≤10字）：0.5-1.5 秒
- 中等消息（11-30字）：1-3 秒
- 长消息（>30字）：2-5 秒

### 3. 频率限制

系统会自动限制发送频率：
- **连续间隔**：两条消息之间至少间隔 10 秒
- **每分钟限制**：每个账号每分钟最多发送 3 条消息
- **每小时限制**：每个账号每小时最多发送 10 条消息（可配置）

### 4. 防封建议

- **新账号**：建议使用保守设置（延迟 5-10 秒，每小时 5 条）
- **老账号**：可以适当放宽限制
- **分散发送**：避免在短时间内集中发送大量消息
- **独立代理**：为议价账号配置独立代理，避免 IP 关联
- **话术多样化**：准备多套话术轮换使用，避免重复内容

## 配置说明

### 基础配置

| 配置项 | 说明 | 必填 |
|--------|------|------|
| 启用飞书聊一聊 | 开启/关闭功能 | 是 |
| App ID | 飞书应用 ID | 是 |
| App Secret | 飞书应用密钥 | 是 |
| Encrypt Key | 事件加密密钥 | 否 |
| 议价账号 | 用于发送消息的闲鱼账号 | 是 |
| 默认议价话术 | 未指定内容时使用的默认消息 | 是 |

### 防封保护配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 最小延迟 | 2 秒 | 进入聊天框后的最短等待时间 |
| 最大延迟 | 5 秒 | 进入聊天框后的最长等待时间 |
| 每小时限制 | 10 条 | 每个账号每小时最多发送条数 |

## 故障排查

### 问题：机器人不响应

**可能原因及解决方案：**

1. **应用未发布**
   - 检查飞书应用是否已发布并通过审核
   - 进入"版本管理与发布"确认应用状态

2. **Webhook 地址配置错误**
   - 检查请求地址是否为正确的公网地址
   - 确保地址格式为 `http://IP:端口/feishu/webhook`
   - 点击"验证"按钮测试连接

3. **事件未订阅**
   - 检查是否订阅了"接收消息"事件
   - 重新添加 im.message.receive_v1 事件

4. **机器人未添加到群聊**
   - 确认机器人已添加到群聊
   - 检查群成员是否有权限@机器人

5. **本系统未启用功能**
   - 检查"启用飞书聊一聊"开关是否打开
   - 点击"测试连接"验证配置

### 问题：消息发送失败

**可能原因及解决方案：**

1. **Cookie 失效**
   - 检查议价账号的 Cookie 是否有效
   - 在账号管理中刷新 Cookie

2. **账号被风控**
   - 检查账号是否被闲鱼限制
   - 登录闲鱼查看是否有异常提示

3. **商品不存在**
   - 检查商品链接是否正确
   - 确认商品未被删除或下架

4. **频率限制触发**
   - 检查是否触发了频率限制
   - 等待一段时间后重试
   - 调整频率限制配置

### 问题：收到消息但无法解析链接

**可能原因及解决方案：**

1. **链接格式不支持**
   - 检查链接是否为闲鱼商品链接
   - 尝试使用完整的商品链接

2. **消息格式错误**
   - 确保@机器人后有空格
   - 链接前后不要有特殊字符

### 问题：配置保存失败

**可能原因及解决方案：**

1. **网络问题**
   - 检查网络连接是否正常
   - 刷新页面后重试

2. **权限不足**
   - 确认当前用户有权限修改配置
   - 联系管理员获取权限

## 安全建议

1. **使用专用账号**：建议使用专门的闲鱼账号进行议价，避免主账号被封
2. **配置独立代理**：为议价账号配置独立代理 IP，避免与其他账号关联
3. **控制发送频率**：不要设置过高的发送频率，建议每小时不超过 20 条
4. **定期更换话术**：准备多套话术轮换使用，避免被系统识别为机器人
5. **监控账号状态**：定期检查议价账号的状态，发现异常及时处理
6. **保护密钥安全**：不要将 App Secret 和 Encrypt Key 泄露给他人
